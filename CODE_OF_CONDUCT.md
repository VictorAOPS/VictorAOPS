# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge
We commit to making participation in this project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment:
- Respectful and constructive feedback
- Professional and inclusive communication
- Accountability and empathy

Examples of unacceptable behavior:
- Harassment, insults, or discriminatory language
- Publishing others' private information without explicit permission
- Unprofessional or disruptive conduct

## Enforcement
Report incidents through GitHub issues (or private contact listed in `SECURITY.md` if sensitive).
Project maintainers will review and respond appropriately.
